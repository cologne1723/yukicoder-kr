# No.1883 SLASH

## 제한

- 시간 제한: 2초
- 메모리 제한: 512MB

## 문제

$0$ 이상 $9$ 이하의 서로 다른 정수 $a, b, c$가 연속으로 주어집니다.

$a, b, c$중 최댓값을 $M$, 최솟값을 $m$이라고 할 때, $M-m$을 구해주세요.

## 입력

|$abc$|
|:-|

- $a, b, c$는 $0$ 이상 $9$ 이하의 서로 다른 정수이다.
- $a, b, c$ 사이에는 공백이 없다.

## 출력

$M-m$을 출력해주세요. 마지막에 개행문자를 출력해주세요.

## 예제

### 예제 1

입력

```
168
```

출력

```
7
```

$M = 8, m = 1$이므로 $M-m=7$입니다.

### 예제 2

입력

```
053
```

출력

```
5
```

$a = 0$일 수도 있습니다.
