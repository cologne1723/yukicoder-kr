# No. 1849 Three Times Value

## 제한

- 시간 제한: 2초
- 메모리 제한: 512MB

## 문제

$111, 141414$처럼, 같은 수를 문자열로 해서 $3$번 이어쓴 수를 **3배수**라고 합니다. 단, $000, 010101$등, $0$으로 시작하는 수는 **3배수**가 아닙니다.
$1$ 이상 $N$ 이하의 수 중, **3배수**는 몇 개 있습니까?

## 입력

|$N$|
|:-|

- $1 \le N \le 10^{15}$

## 출력

답을 출력해주세요.

## 예제

### 예제 1

입력

```
400
```

출력

```
3
```

$111$, $222$, $333$이 조건을 만족하므로, 답은 $3$입니다. 그러므로, $3$을 출력합니다.

### 예제 2

입력

```
12
```

출력

```
0
```

조건을 만족하는 수는 없습니다.

### 예제 3

입력

```
77777777777
```

출력

```
999
```
